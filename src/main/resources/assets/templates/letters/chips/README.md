# Overview
This folder is organised into different sections according to letter type, each of these is shown below along with a description of what each folder represents.
Each folder will also contain version numbers starting with v1.0


| Folder name | Description of letter |
|-------------|-----------------------|
|CSIDVDEFLET  |Overdue confirmation statement and director identity verification statements |
|IDV_NEW_DIRECTIONS_LETTER |Inform new PSCs that they need to verify their identity |
|IDV_TRANSITIONAL_DIRECTIONS_LETTER |Inform existing PSCs that they need to verify their identity |
|IDVPSCDEFAULT|Overdue identity verification statement for your role as a person with significant control (PSC) |
|IDVPSCEXT1   |Acceptance of a first extension of the identity verification deadline for a PSC |
|IDVPSCEXT2   |Acceptance of a second extension of the identity verification deadline for a PSC |
